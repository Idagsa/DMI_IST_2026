# DMI_IST_2026

Code for project revolving using ML to determine the GrIS IST.

IST_retrievals.py supplied by gd@dmi.dk and maiwin@dtu.dk

[Remita, 2025] supplied by zoe.remita@gmx.fr

The [plots] folder contains variables and commands used for creating readable and consistent plots, following DTU's (Technical University of Denmark) colour code

PROMICE_hist.ipynb is made as an overview of the PROMICE data used in the project.
ML_final.ipynb trains, test and validates an NGBoost model on provided data.
